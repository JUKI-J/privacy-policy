OneSync 개인정보처리방침

시행일: 2025년 1월 1일 | 개정일: 2025년 1월 1일

📌 핵심 요약
OneSync는 최소한의 정보만 수집하며, 전송된 데이터는 7일 이내에 자동 삭제됩니다. 현재는 회원가입이 필요 없으며, 광고나 추적을 하지 않습니다.

1. 개인정보의 수집 및 이용 목적

OneSync(이하 "서비스")는 사용자의 프라이버시를 최우선으로 생각하며, 서비스 제공에 필요한 최소한의 정보만을 수집합니다.

1.1 수집하는 정보

  • 기기 정보: 기기 고유 식별자(UUID), 기기 모델명, 운영체제 버전
  • 세션 정보: 세션 코드, 참여 시간, 연결 상태
  • 전송 데이터: 사용자가 공유하는 파일 및 텍스트 (암호화되어 임시 저장)

1.2 수집하지 않는 정보

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보
  • 연락처, 캘린더 등 기기 내 개인 데이터
  • 사용 행태 추적 정보

2. 개인정보의 수집 방법

서비스는 다음과 같은 방법으로 정보를 수집합니다:

  • 앱 최초 실행 시 자동 생성되는 기기 UUID
  • 세션 생성 또는 참여 시 필요한 연결 정보
  • 사용자가 직접 선택하여 공유하는 파일 및 텍스트

3. 개인정보의 이용

수집 항목 이용 목적 보관 기간
기기 UUID 세션 내 기기 식별 세션 종료 후 7일 이내 삭제
세션 정보 기기 간 연결 유지 세션 종료 후 7일 이내 삭제
전송 데이터 다른 기기로 전달 전송 완료 후 7일 이내 삭제

4. 개인정보의 보호

4.1 보안 조치

  • 종단간 암호화: 모든 데이터는 AES-256으로 암호화되어 전송
  • SSL/TLS: 모든 통신은 HTTPS를 통해 보호
  • 임시 저장: 서버에 영구 저장되는 데이터 없음
  • 자동 삭제: 모든 데이터는 7일 이내 자동 삭제
  • 클라우드 스토리지: Cloudflare R2를 통한 안전한 파일 관리

4.2 데이터 저장 위치

임시 데이터는 독일(Hetzner 데이터센터)에 위치한 서버에 저장되며, 최대 7일간 보관 후 자동 삭제됩니다. 파일 저장은 Cloudflare R2를 통해 관리됩니다.

5. 개인정보의 제3자 제공

서비스는 원칙적으로 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다:

  • 법령의 규정에 의한 경우
  • 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

6. 이용자의 권리

사용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다:

  • 세션 종료: 세션을 종료하면 활성 연결이 종료되며, 데이터는 7일 이내 자동 삭제됩니다
  • 앱 삭제: 앱을 삭제하면 기기에 저장된 모든 로컬 데이터가 제거됩니다
  • 선택적 권한: 파일 접근 등의 권한은 선택적으로 허용/거부 가능합니다

7. 쿠키 및 추적 기술

서비스는 다음과 같은 정책을 따릅니다:

  • 웹 버전에서 세션 유지를 위한 필수 쿠키만 사용
  • 광고 또는 분석을 위한 추적 쿠키 사용 안 함
  • 제3자 추적 도구 사용 안 함

8. 아동의 개인정보 보호

서비스는 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 만 14세 미만 아동이 서비스를 이용하려면 부모 또는 법정대리인의 동의가 필요합니다.

9. 개인정보처리방침의 변경

이 개인정보처리방침은 법령, 정책 또는 서비스 변경에 따라 수정될 수 있습니다. 변경 시 앱 내 공지사항을 통해 사전 고지하며, 중요한 변경사항은 30일 전에 고지합니다.
